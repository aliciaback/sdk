# 갈라체인 SDK

갈라체인 개발에 오신 것을 환영합니다! 
갈라체인은 Web3 게임, 엔터테인먼트 등의 기반이 되도록 설계된 레이어 1 블록체인입니다.

## 기능 

- 원활한 체인코드 개발을 가능하게 하는 유틸리티 라이브러리
- 코드 핫 리로드와 로컬 블록 브라우저 기능을 갖춘 로컬 개발 환경
- 체인코드 템플릿을 이용한 쉬운 시작
- 갈라체인과의 통합

자세한 내용은 다음을 참고하세요: [갈라체인](galachain.md).
